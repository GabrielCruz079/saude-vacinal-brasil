# 📊 Projeto de Análise de Vendas

Projeto desenvolvido com foco em análise de dados utilizando Python, MySQL e Power BI.

## 🚀 Tecnologias utilizadas

* Python
* Pandas
* MySQL
* Power BI
* SQL
* Excel/CSV

---

## 📁 Estrutura do Projeto

* Extração de dados com Python
* Consultas SQL no MySQL
* Tratamento e análise com Pandas
* Dashboard interativo no Power BI

---

## 📊 Principais análises

* Faturamento total
* Lucro por produto
* Produtos mais vendidos
* Vendas por cidade
* Margem de lucro
* Performance comercial

---
